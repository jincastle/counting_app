- 초기화 버튼을 클릭 시, 숫자를 0으로 변경하기

layout_weight="0dp"가 있는 이유 : Vetical(세로) 방향에서 layout_weight를 사용하려면 layout_height 값을 0dp로 설정해야하는 규칙이 있기 때문입니다.
